# 宠物领养管理系统 java1143

## 项目概述

宠物领养管理系统是一款基于前后端分离架构的信息管理系统，旨在提供宠物领养相关的各项服务，包括流浪动物管理、领养申请处理、宠物信息发布等。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 系统角色

#### 管理员

- **前台操作**: 涵盖所有用户功能
- **后台操作**:
  - 用户管理
  - 角色管理
  - 菜单管理
  - 文件管理
  - 流浪动物管理
  - 待绝育动物管理
  - 申请领养管理
  - 评论管理
  - 流浪动物救助
  - 喂养点管理
  - 动物走失管理
  - 救助站管理
  - 帖子管理
  - 捐赠管理
  - 公告管理
  - 科普文章管理
  - 活动管理

#### 用户

- 首页浏览
- 宠物领养
- 流浪宠物救助
- 宠物喂养点查询
- 丢失宠物查看

- 宠物论坛交流
- 宠物捐赠
- 科普文章阅读
- 热门活动参与
